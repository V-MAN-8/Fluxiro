The ultimate CLI tool for downloading videos from any platform**

That is a lightweight, lightning-fast command-line tool designed to help you download videos  from any platform with ease. 

---

## Features

- **Multi-Platform Support:** Download from YouTube, Vimeo, Twitter (X), Instagram, TikTok, and more.
- **High Quality:** Download the best quality (depending on source).
- **Batch Processing:** multiple links file to download in bulk.
